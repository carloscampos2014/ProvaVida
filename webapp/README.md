# ProvaVida WebApp

Este diretório conterá o frontend do sistema ProvaVida, desenvolvido em React + TypeScript.

## Stack Inicial
- React 18+
- TypeScript
- Vite (build rápido)
- ESLint + Prettier (qualidade de código)
- Integração futura com API REST ProvaVida

## Como iniciar

1. Instale Node.js (18+)
2. Instale as dependências (após inicialização do projeto):
   ```bash
   cd webapp
   npm install
   ```
3. Para rodar o frontend (após setup):
   ```bash
   npm run dev
   ```
